## 面試技巧複習

### Technical Interview 技巧

1. **仔細聽題**：確認理解，問清楚 edge cases
2. **大聲思考**：讓面試官了解思考過程
3. **先討論方法**：不要急著寫 code
4. **寫完測試**：用幾個 test case 驗證
5. **分析複雜度**：說明時間與空間複雜度
6. **保持溝通**：卡住時主動討論

### Behavioral Interview 技巧

1. **準備充分**：每個故事都練習過
2. **量化成果**：用數字說話
3. **保持真實**：不要編造故事
4. **展現成長**：說明學到什麼
5. **正面態度**：即使談失敗也要積極
